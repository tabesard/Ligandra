#   [boost]  xgboost, lightgbm                -> gradient-boosted classical models
#   [chembl] chembl_webresource_client        -> live ChEMBL queries
#   [dock]   meeko (+ external vina binary)   -> docking
# The bundled demo config (examples/local_experiment.yaml) only needs
# ecfp/lipinski featurizers, ridge/random_forest models, and the graph_ga
# generator — all covered by the core deps in pyproject.toml. Install extras
# locally with, e.g.:
#   pip install -e ".[deep,gnn,boost,chembl,dock,tracking]"
